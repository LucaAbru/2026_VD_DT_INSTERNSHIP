# 2026_VD_DT_INSTERNSHIP

This repository contains all models developed during the 2026 Design Team Internship.

## Repository Structure

### Documentation
Contains the methodologies, assumptions, and theory behind the developed models.

### Ride Models
Contains all suspension and vehicle dynamic models, including:
- Quarter-car model
- Half-car model
- 7 DOF full vehicle model

### Lap Sim
Contains the quasi steady-state lap simulation tools and supporting scripts.
